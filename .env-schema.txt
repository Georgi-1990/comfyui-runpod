## Optional
COMFY_UPDATE

Type: boolean
Default: false

Обновляет ComfyUI и установленные custom nodes при запуске контейнера.
* true — выполняется git pull
* false — используется текущая версия в /workspace

Рекомендация:
Включать только при необходимости, т.к. увеличивает время старта.

CUSTOM_NODES
* Type: string (comma-separated)
* Default: (empty)

Список кастомных нод для установки.
Поддерживаемые значения:

manager
impact-pack
controlnet

Пример:
CUSTOM_NODES=manager,impact-pack,controlnet

Ноды устанавливаются в:
/workspace/ComfyUI/custom_nodes

## JUPYTER_PASSWORD
* Type: string
* Default: (empty)

Токен доступа для JupyterLab (:8888).
Если не задан — Jupyter стартует без токена.

## PUBLIC_KEY
* Type: string
* Default: (empty)

SSH public key для root-доступа.
 Если не задан:
  * генерируется случайный пароль
  * пароль выводится в логах контейнера
  
## Advanced (build-time, не для UI)
Эти параметры НЕ рекомендуется выносить в RunPod UI,
они задаются при сборке Template:
| ARG            | Назначение                   |
| -------------- | ---------------------------- |
| `CUDA_VERSION` | Версия CUDA runtime          |
| `TORCH_CUDA`   | CUDA build PyTorch (`cu124`) |
| `BASE_IMAGE`   | `runtime` / `devel`          |

## Recommended Default UI Configuration
* COMFY_UPDATE=false
* CUSTOM_NODES=
* JUPYTER_PASSWORD=
* PUBLIC_KEY=

## Поведение при старте
1. Проверка CUDA / GPU
2. Установка ComfyUI (если отсутствует)
3. Установка custom nodes (если заданы)
4. Запуск ComfyUI на 8188
